## What this is
- WSL X11 helper script and documentation.
- RDP entry for connecting to the Windows host.
- Codex history and skills snapshots for traceability.
- CI checks to keep required assets present.

## Quick start
- WSL X11: `source /home/standard/scripts/wsl-x11.sh`
- RDP: open `docs/windows-host.rdp` in your preferred client.
